Switchless
------
A modern osu! server switcher using peppy's "devserver" startup variable.

Using Switchless
------
```sh
1. Get the latest version of Switchless from https://ainu.pw/static/Switchless.exe.
2. Run Switchless.exe.
3. Click the "Select osu!.exe" button.
4. Select your osu!.exe file.
5. Click the "Start osu!" button.
```
